✏️ What are Media Queries?

We can use media queires to apply different styles to our website, depending on how the website is being viewed. For example, if our website is being viewed on mobile, we might want to make all of the margins and padding smaller, hide our navigation completely behind a burger icon. This is called responsive design

Mobile first design is common in modern web dev, websites are most commonly viewed on mobile devices

Most often, we care about the screen width in pixels. For example, the media query below asks the browser "How wide are you right now?", and will only make the h1s green if the screen width is under 1000 pixels

❗ Be reasonable ❗

Responsive design is one of the most time consuming parts of web dev.

For your project, you must include media queries, but you only need to design for two screen widths.

i.e. design your project on desktop, and then produce a single query that changes many styles when the width is below a certain width
