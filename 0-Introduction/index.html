✨Syntactically Awesome Style Sheets (SASS)

SASS is a CSS pre-processor that makes writing styles much more natural for programmers.

It adds many features, like calculations, variables, and nested structures, and allows you to split up your files in a more digestable way.

When you read through the code for this page, you will notice that there are a number of .SCSS files, and only one .CSS file. I wrote all of the code in SCSS, and used a compiler to translate the code into a regular style sheet for distribution.
