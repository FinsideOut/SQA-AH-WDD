✨Sassy Stylesheets (SCSS)

SCSS is a CSS pre-processor that makes writing stylesheets much more natural for programmers.

It adds many features to "vanilla" CSS, like calculations, variables, and nested structures, and allows you to split up your files in a more digestable way.

Image from Kasun Weerasinge's article on Medium

When you read through the code for this page, you will notice that there are a number of SCSS files, and only one CSS file. I wrote all of the code in SCSS, and used a compiler to translate the code into a regular stylesheet for distribution.
