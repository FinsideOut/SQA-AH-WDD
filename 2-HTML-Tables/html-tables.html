Later, we will be using tables to format data that we have retrieved from a database using PHP, but for now... just tables.

❗ Just for Data ❗

Tables should only be used for displaying data.

Once you get the hang of them you might be tempted to use them to format a navigation bar, or a footer.

Don't.

Modern browsers have many accessibillity features built in. Blind people can access websites by using an eReader. This is why semantic HTML is important - properly structured HTML makes your website easier to read with modern tools. Don't be lazy!
